## Definition

Rate Control is the strategy of managing [[concepts/Atrial Fibrillation]] symptoms by controlling ventricular rate, as an alternative to rhythm control via [[concepts/Cardioversion]].

## Related

- See [[concepts/Atrial Fibrillation]] for the dominant indication
- See [[concepts/Cardioversion]] for the alternative strategy
- See [[concepts/Beta Blocker]] for rate-control agents
