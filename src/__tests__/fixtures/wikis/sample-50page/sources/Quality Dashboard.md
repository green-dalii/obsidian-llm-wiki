## Source

Synthetic quality metrics dashboard for the fictional Pinewood Heart Center. Used as a graph anchor for entities and concepts linked to outcome measurement.

## Metrics

Synthetic metrics include:

- 30-day readmission rate (heart failure cohort)
- Door-to-balloon time (STEMI cohort)
- Anticoagulation adherence (atrial fibrillation cohort)
- LDL target attainment (post-MI cohort)
- Bleeding event rate (anticoagulation cohort)
